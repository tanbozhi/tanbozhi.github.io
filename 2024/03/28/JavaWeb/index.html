Maven

Maven是apache旗下的一个开源项目，是一款用于管理和构建Java项目的工具。

作用

1、依赖管理

方便快捷的管理项目依赖的资源（jar包），比卖你版本冲突问题。

2、统一项目结构

提供标准、统一的项目结构

3、项目构建

标准跨平台(Linx、Window、Mac OS)的自动化项目构建方式

仓库

用于存储资源，管理各种jar包

先查询本地，在查询私服在，在查询中央仓库

本地仓库：自己计算机上的一个目录

中央仓库：由maven团队维护的全球唯一的。仓库地址：https://repo1.maven.org/maven2/

远程仓库：（私服）一般由公司团队搭建的私有仓库。

安装与配置

依赖配置

依赖：指当前项目运行所需要的jar包，一个项目中可以引入多个依赖

依赖范围

生命周期

对所有的maven项目构建过程进行抽象和统一

maven中有3套独立的生命周期

每套生命周期包含一些阶段，阶段是有顺序的，后面阶段依赖于前面的阶段

在同一套生命周期中，当运行后面阶段时，前面的阶段都会运行

MySQL数据模型

关系型数据库(RDBMS)：建立在关系模型基础上，由多张相互连接的二维表组成的数据库

特点：

使用表存储数据，格式统一，便于维护

使用SQL语言操作，标准统一，使用方便，可用于复杂查询

SQL简介

SQL：一门操作关系型数据库的编程语言，定义操作所有关系型数据库的统一标准。

SQL语句可以单行或者多行书写，一分号结尾。

SQL语句可以使用空格/缩进来增强语句的可读性

MySql数据库的SQL语句不区分大小写

注释：

​ 单行注释：-- 注释内容 或者 # 注释内容(Mysql独有)

​ 多行注释：/* 注释内容 */

数据库操作
表操作(创建)

约束

概念：作用于表中字段上的规则，用于限制存储在表中的数据

目的：保证数据库中数据的正确性，有效性和完整性

约束 描述 关键字
非空约束 限制该字段不能为null not null
唯一约束 保证字段的所有数据都是唯一的，不重复的 unique
主键约束 主键是一行数据的唯一标识，要求非空且唯一。可选：自增 primary key auto_increment
默认约束 保存数据时，如果未指定该字段值，则采用默认值 default
外键约束 让两张表的数据建立连接，保证数据的一致性和完整性 foreign key

4.组件与组件属性

组件：包含内容，样式和功能。

首字母必须大写

函数组件

必须返回一段html(本质是一个react对象)

调用：<MyFunc/>

类组件

必须继承React.Component

必须提供render函数，用于渲染组件

组件的属性

属性可以传任何数据（html也可以传递）

函数组件

属性会作为一个对象属性，传递给函数的参数

属性的值是不能更改的(单向数据流)

类组件

属性会作为一个对象属性，传递给构造函数的参数

组件状态

组件可以自行维护的数据

组件状态仅在类组件中有效

状态初始化（两种方法）
改变状态

使用setState()

setState改变状态可能是异步的

如果改变状态的代码处于某个html元素中，则是异步的，否则就是同步

另一种写法

第一个参数是一个函数，函数的参数是当前的状态，该函数的返回结果会覆盖之前的状态，也是异步执行

多次调用会将函数放到一个队列中，顺序调用

事件

生命周期

仅存在类组件中

旧：React < 16.0.0
新：React >= 16.0.0

删除了：componentWillMount,componentWillReceiveProps,componentWillUpdate

传递元素内容

父组件给子组件传递元素内容

表单

受控组件

组件的使用者可以完全控制该组件的行为和内容，通常情况下，受控组件没有自身的状态，其状态完全受到属性的控制

表单上有value和onChange事件后，才是受控组件

非受控组件

组件的使用者没有能力控制该组件的行为和内容，组件的行为和内容完全自行控制

属性默认值
